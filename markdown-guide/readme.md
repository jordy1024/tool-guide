## 数学公式或符号的添加
- n次方   

举例：     
2的3次方，2<sup>3</sup>      
`2<sup>3</sup>`     
-2的9次方，-2<sup>9</sup>     
`-2<sup>9</sup>`      

- 上下标

x的下标 x<sub>2</sub>

```
x<sub>2</sub>
```
